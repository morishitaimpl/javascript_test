# JavaScript学習 - 初心者向けサンプル

JavaScriptを初めて学ぶ方向けの学習用リポジトリです。

## 📚 学習内容

このプロジェクトでは、以下のJavaScriptの基本概念を学習できます：

1. **Hello World** - 基本的な出力
2. **変数の使い方** - 文字列、数値、真偽値、配列
3. **計算** - 基本的な算術演算
4. **ユーザー入力** - フォームからの入力処理
5. **条件分岐** - if文を使った判定
6. **繰り返し処理** - for文を使ったループ
7. **配列** - 複数の値の管理

## 🚀 使い方

1. `index.html` をブラウザで開いてください
2. 各セクションのボタンをクリックして、JavaScriptの動作を確認してください
3. 開発者ツール（F12キー）を開いて、コンソールメッセージも確認してみてください

## 📁 ファイル構成

- `index.html` - メインのHTMLファイル（ブラウザで開くファイル）
- `script.js` - JavaScriptのコード（学習用サンプル）
- `README.md` - このファイル（説明書）

## 💡 学習のポイント

- **実際に動かしてみる**: ボタンをクリックして結果を確認
- **コードを読む**: `script.js` を開いて、どのように動いているか確認
- **コンソールを確認**: 開発者ツールでコンソールメッセージを見る
- **値を変更してみる**: 入力欄に異なる値を入れて試す

## 🔧 開発者ツールの使い方

1. ブラウザでF12キーを押す
2. 「Console」タブを選択
3. JavaScriptのメッセージやエラーを確認できます
